# Organizing and Summarizing Air Quality Data

## Project Overview
With the use of AirPurple, UNMC has been able to monitor air conditions from February 2025 to March 2025 but has not been able to translate the data. With the data collected, I have been instructed to manipulate the data’s organizational structure to help better understand and simplify its readings.
